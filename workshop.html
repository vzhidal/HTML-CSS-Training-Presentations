Meta tags

  • Provide metadata about the HTML document.

  • Can be used by browsers, search engines, or other web services.

Meta tag attributes

charset
declares the character encoding used of the page
content
gives the value associated with the http-equiv or name attribute, depending of the context
http-equiv
defines the pragma that can alter servers and user-agents behavior.
name
defines the name of a document-level metadata.

Css Basics

reset vs. normalize

reset
  • Removes all built-in browser styling.
  • Standard elements have no decoration at all.
normalize
  • Makes built-in browser styling consistent across browsers.
  • Standard elements appear in a consistent way across browsers.
